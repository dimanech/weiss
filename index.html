About

Weiss is HTML5-CSS3 blank canvas sass-middleman helper for fast-start development. Based around tasks and problems that arise with me every day on my work - so in some cases it is not universal or full production ready framework.

Main features:

  • Not a prototyping tool like Bootstrap or Foundation, just blank canvas
  • Middelman/sass/compass
  • BEM structure of blocks
  • HTML5 semantic markup
  • CSS3 ready with mixins for working with vendor prefixes
  • Media Query structure and mixins
  • WAI ARIA support
  • Hg/Git -ignore
